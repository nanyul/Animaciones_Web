Animaciones CSS

¿En qué consiste?

CSS proporciona un modelo declarativo para definir animaciones, es decir, el desarrollador describe qué debe suceder y el navegador se encarga de ejecutarlo. Las animaciones en CSS utilizan la regla @keyframes para definir los estados intermedios del movimiento.

Ejemplo de uso de @keyframes

@keyframes girar {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}

.rueda {
    animation-name: girar;
    animation-duration: 3s;
    animation-timing-function: linear;
    animation-iteration-count: infinite;
}

En este caso, el elemento con clase .rueda gira constantemente sobre su eje, repitiendo el ciclo indefinidamente.

A continuación, las propiedades clave de la animación

Propiedades Clave de la Animación

animation-name

Nombre de la animación definida con @keyframes.

animation-duration

Tiempo total de ejecución.

animation-timing-function

Define la curva de velocidad del movimiento.

animation-delay

Tiempo antes de comenzar.

animation-iteration-count

Número de repeticiones.

animation-direction

Sentido de reproducción (normal, reverse, alternate).

animation-fill-mode

Estado al inicio/final (none, forwards, backwards, both).

Transiciones

Las transiciones son animaciones más simples, que se activan al producirse un evento (como hover o focus).

Ejemplo

button {
  background-color: #007bff;
  transition: background-color 0.3s ease;
}

button:hover {
  background-color: #0056b3;
}

En este caso, el botón cambia suavemente de color al pasar el cursor por encima, lo que genera una sensación de respuesta inmediata.
